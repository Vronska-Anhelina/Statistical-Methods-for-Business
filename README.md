# Statistical-Methods-for-Business
A comprehensive collection of statistical analysis projects focused on hypothesis testing, experimentation, and data-driven decision-making.

# Core Modules:

1. Experimentation & A/B Testing
Sample Size Estimation: Calculating required traffic to achieve statistical power.

Significance Testing: Z-tests and T-tests to evaluate conversion rate shifts and CTR performance.

Non-Parametric Analysis: Applying Mann-Whitney U tests for non-normally distributed web data.

2. Descriptive & Inferential Statistics
Distribution Analysis: Testing for normality and identifying data skewness in salary and traffic datasets.

Confidence Intervals: Estimating population parameters with defined margins of error.

Correlation Research: Analyzing dependencies between variables (e.g., Bill vs. Tip correlation).

3. Business Case Studies
Loyalty Program Evaluation: Statistical assessment of customer retention initiatives.

Sales Performance: Comparative analysis of brands and revenue streams.

# Technologies:

Python, SciPy, Statsmodels, Pandas, Matplotlib, Seaborn
